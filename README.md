# 🎯 Quiz App

An interactive web-based Quiz Application with timer, student login, answer validation, and leaderboard.

---

## 🚀 Features

- ⏱️ Timer-based questions
- 👤 Student login (Name & Roll Number)
- ✅ Instant feedback (Correct / Wrong Answer)
- 📊 Score calculation with percentage
- 🏆 Leaderboard system
- 💾 Data stored using browser localStorage
- 🌐 Works without backend (frontend-only)

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
